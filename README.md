# Better Section Word Count

This plugin is the same as the **Better Word Count** plugin, except that BSWC can count the current section words, characters, etc.

本人不会逆向，所以上传了Gemini帮我改好的main.js文件，现在插件已经可以统计目前所在章节数据了。
